# Site Leticia — Ménage Bordeaux

Site vitrine statique pour présenter un service de ménage à domicile à Bordeaux.

Pages :
- `index.html` : accueil, présentation rapide, portrait et zone d’intervention.
- `prestations.html` : prestations avec boutons qui pré-remplissent la page estimation.
- `apropos.html` : présentation humaine et prudente.
- `avis.html` : avis clients + formulaire d’avis WhatsApp.
- `devis.html` : formulaire d’estimation WhatsApp.
- `contact.html` : page contact séparée.
- `mentions-legales.html` : mentions légales à compléter avec les informations administratives exactes.
- `confidentialite.html` : politique de confidentialité simple.

À modifier avant publication commerciale définitive :
1. Remplacer `WHATSAPP_NUMBER = "33768345608"` dans `script.js` par le vrai numéro WhatsApp de Leticia.
2. Compléter les mentions légales avec les informations administratives exactes.
3. Remplacer les avis exemples par de vrais avis avec accord.
4. Optimiser les images si nécessaire pour améliorer le chargement mobile.
